## Introduction

투표기반으로 가장 일반적인 조합이나 행동양식을 찾아 통계를 내는 웹기반 서비스. 소위 국룰을 정리하는 사이트를 만들기 위해 프로젝트를 시작했다. 최소한의 개인정보를 토대로 지역/나이/성별/시간 에 따라 트렌드도 트래킹 가능한게 목적.
